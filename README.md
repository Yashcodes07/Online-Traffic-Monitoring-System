# Helmet Detection using YOLOv8

## 📌 Overview
This project detects whether a person is wearing a helmet or not using a YOLOv8 object detection model helpfull in Online Traffic-Monitoring System.

## 🚀 Model Performance
- mAP@0.5: **94.6%**
- mAP@0.5:0.95: **73.9%**
- Inference Speed: **~5ms per image**

## 🧠 Tech Stack
- Python
- YOLOv8 (Ultralytics)
- PyTorch
- OpenCV
- FiftyOne (dataset preparation)
- Google Colab (training)
